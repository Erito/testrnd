USE titusrnd_db
SOURCE D:/kuliah/titusrnd_db.sql;  (kalau ini salin path .sqlnya, klik kanan 'copy as path')


2. Open folder TitusRND (atau klik kanan open terminal, lalu ketik code . pada terminal)
